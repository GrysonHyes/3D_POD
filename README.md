# Description
Proper Orthogonal Decomposition (POD) code written in Python to be used for datasets with 3 dimension (2 spatial dimensions and 1 temporal dimension).

# Demos
The synthetic demo generates data using the data generation script for a rotating pair of vortices.

The real data is Schlieren data from an overexpanded jet. I only attached 50 images of it here, but if you want more for any reason, feel free to reach out.

# Requirements

For the main pod functions:
- numpy

For things like plotting and grabbing files:
- glob
- Pillow
- matplotlib

Probably a beefy computer for larger datasets.
